# FIFA Prediction App

A Machine Learning-based web application that predicts the outcomes of FIFA World Cup 2026 matches using historical football data.  
Built with Python, Streamlit, Scikit-learn, Pandas, and Matplotlib.

## Overview

This project applies data preprocessing, feature engineering, and machine learning classification models (like Logistic Regression and Random Forest) to predict FIFA match outcomes.  
It also includes a Streamlit dashboard to visualize predictions interactively.

---

## Features

- Scrapes and cleans FIFA historical match data  
- Generates EDA plots (heatmaps, feature distributions, etc.)  
- Trains and evaluates multiple ML models  
- Displays accuracy, confusion matrices, and ROC curves  
- Interactive prediction web app using Streamlit
